# 经验总结

| 类型     | 描述                                                                           |
|:-------|:-----------------------------------------------------------------------------|
| DB     | 数据库不支持表的事务管理 （可通过begin ，rollback测试）                                          |
| Thread | java多线程中，子线程设置为守护线程时需注意（比如在写文件时，main方法执行，当main执行完子线程被回收，照成数据未写入文件 参考Logger类） |
